# MugiSideBrowser 設計書

## 1. 概要
`MugiSideBrowser` は、Windows のデスクトップ端に常駐し、Microsoft Edge のサイドバーのように他のウィンドウと重ならずに Web コンテンツを表示できる .NET ベースのサイドバー型ブラウザです。

---

## 2. システム構成・使用技術

- **フレームワーク**: .NET 8/9 (WPF / Windows Presentation Foundation)
- **ブラウザエンジン**: Microsoft Edge WebView2 (Chromium ベース)
- **デスクトップ統合**: Win32 API (`SHAppBarMessage`, `MonitorFromWindow`, `MoveWindow` 等) を使用した Appbar 実装
- **データ保存**: `%APPDATA%\MugiSideBrowser\` 配下に JSON 形式で保存
  - お気に入りデータ: `bookmarks.json`（二重書き込み防止セマフォ制御、テンポラリファイルを介したアトミック保存）
  - ユーザー設定データ: `settings.json`（テーマや配置情報）
- **開発言語**: C# 12 / XAML

---

## 3. 主要機能

### 3.1 ウィンドウ表示モード (Window Display Modes)
デスクトップでの常駐スタイルに合わせて、以下の3つのモードを切り替え可能です。

- **常時表示モード (AppBar)**
  - Windows の AppBar として登録され、デスクトップの有効領域（Work Area）を自動調整（他のウィンドウが重ならないよう押し出す）。
  - 画面の右端または左端に固定表示。
  - タイル状にモニター縦幅いっぱいにフィットし、タスクバーには表示されません（ツールウィンドウスタイル）。
  - 内側の境界線をドラッグすることで、幅（300px〜800px）を自由にリサイズできます。
- **自動隠しモード (AutoHide)**
  - 普段は画面端に数ピクセル（2px）の細い線として隠れます。
  - マウスカーソルが画面端のトリガーゾーン（5px以内）に入ると、スライドインアニメーション（CubicEase Out, 200ms）で元のサイズまで現れます。
  - マウスがウィンドウ外に出ると自動でスライドアウトして隠れます。
- **自由配置モード (Normal/Floating)**
  - AppBar 登録を解除し、通常のフローティングウィンドウとして動作します。
  - タスクバーに表示され、最前面表示も解除されます。
  - 左右および下部のリサイズグリップで自由なサイズ変更が可能です。

### 3.2 分割画面表示機能 (Split Screen)
限られたサイドバーの領域を有効活用するため、最大 3 画面の分割レイアウトに対応しています。

- **画面レイアウト構成**
  - **1画面 (Single)**: メイン（上）ペインのみ表示。
  - **2分割 (Double)**: 上＋中ペイン、または上＋下ペインを表示。
  - **3分割 (Triple)**: 上＋中＋下ペインをすべて表示。
- **ペインの入れ替え (Drag & Drop)**
  - 各ペインのヘッダーをドラッグ＆ドロップすることで、ペイン同士の位置（表示内容）を簡単にスワップできます。
- **インタラクティブな分割制御**
  - 各ペイン間は `GridSplitter` で仕切られており、ドラッグによる高さの動的な割り振りが可能です。
  - アクティブなペインは、ヘッダー左側のブランドカラーのインジケーター（丸印）および太字テキストで視覚的に強調され、アドレスバーの URL 表示や操作（戻る／進む／更新）と連動します。

### 3.3 お気に入り／垂直タブ機能 (Favorites & Vertical Tabs)
本アプリでは「お気に入り」がブラウザの「タブ」を兼ねる設計となっています。

- **垂直タブリスト**
  - サイドバー左側（または右側）のドック領域に、お気に入りのファビコンが縦一列に並びます。
  - ドラッグ＆ドロップによって、お気に入りの表示順を視覚的に並び替えられます。
  - 区切り線（Separator）を追加してお気に入りを整理できます。
- **3つのステータスインジケーター**
  1. **アクティブ状態 (Active)**: ファビコン左側に「青い縦バー」を表示。現在フォーカスされているアクティブペインに表示されているお気に入りを示します。
  2. **オープン状態 (Open)**: ファビコン左側に「緑の縦バー」を表示。分割画面のいずれかのペインに表示中のお気に入りを示します。
  3. **メモリロード状態 (Loaded)**: ファビコンの右下に「緑の丸印」を表示。WebView2 インスタンスが生成され、メモリ上にページがロードされていることを示します。
- **ファビコンの自動取得**
  - 読み込み時に WebView2 から `FaviconUri` を直接取得。
  - 取得できない、または未ロードの場合は DuckDuckGo の Favicon 取得API (`https://icons.duckduckgo.com/ip3/{domain}.ico`) を介して自動フェッチします（フォールバック付き）。

### 3.4 メモリ節約／スリープ機能 (Tab Sleep)
複数ペインや多数のお気に入りを開いた際のメモリ（RAM）消費を最小限に抑える仕組みを備えています。

- **タブのスリープ (Unload)**
  - お気に入り項目を右クリックして「タブを閉じる」を選択すると、そのページの WebView2 インスタンスが破棄され、メモリが完全に解放されます（Loaded インジケーターが消灯）。
- **スリープ画面プレースホルダー**
  - メモリ解放されたお気に入りがペインに割り当てられている場合、ブラウザ領域には「スリープ状態」であることを示す専用のプレースホルダー画面が表示されます。
  - プレースホルダーの「再開」ボタンまたは画面全体のクリックにより、必要な時だけ WebView2 インスタンスをオンデマンドに再生成してページを再ロード（レジューム）できます。

## 4. UI/UX デザイン

- **モダンなテーマシステム**
  - **ダークモード / ライトモード**: ユーザー設定に応じてリアルタイムに切り替え可能。
  - Fluent Design を意識した、半透明ホバーやスタイリッシュなコントロールスタイルを採用。
- **多言語対応 (Localization)**
  - 日本語（`StringResources.ja.xaml`）とデフォルトの英語（`StringResources.xaml`）の ResourceDictionary を持ち、システム設定に応じて動的に文言が切り替わります。
- **心地よいマイクロアニメーション**
  - 垂直タブのファビコンにマウスホバーすると、スムーズな Storyboard アニメーションによってアイコンが 1.15倍 にスケールアップし、インタラクティブな手応えを与えます。
  - ウィンドウのスライドイン／アウトやリサイズ時のスムーズな追従。

## 5. デスクトップ統合とその他の便利機能

- **多重起動防止＆ウィンドウ復元**
  - アプリの多重起動を防止し、2番目のインスタンスが起動された際には独自のウィンドウメッセージ (`MugiSideBrowser_ShowWindowMessage`) を送信して、すでに起動している既存のサイドバーウィンドウをフォアグラウンドに復元します。
- **システムトレイ（タスクバー通知領域）常駐**
  - 最小化または閉じる操作の代わりにシステムトレイに常駐し、タスクバーを占有しません。トレイアイコンのダブルクリックで即座にサイドバーが復帰します。
- **ユーザーエージェント (User-Agent) 切り替え**
  - メニューからワンクリックで「モバイル表示」と「デスクトップ表示」を切り替え可能です。WebView2 インスタンスの User-Agent 設定を動的に書き換えて再読み込みします。
- **外部リンクの挙動制御**
  - WebView2 内で新しいウィンドウ（`target="_blank"` や Ctrl＋クリック等）が要求された際、標準の Web ブラウザで開くかどうかを設定メニューから制御できます。
