# MugiSideBrowser 設計書

## 1. 概要
`MugiSideBrowser` は、Windows のデスクトップ端に常駐し、Microsoft Edge のサイドバーのように他のウィンドウと重ならずに Web コンテンツを表示できるサイドバー型ブラウザです。

## 2. システム構成
- **フレームワーク**: .NET 8/9 (WPF)
- **ブラウザエンジン**: Microsoft Edge WebView2
- **デスクトップ統合**: Win32 API (`SHAppBarMessage`) を使用した AppBar 実装
- **言語**: C#

## 3. 主要機能

### 3.1 AppBar 統合 (サイドバー動作)
- **デスクトップ端への固定**: 画面の右端または左端に固定して表示。
- **ワークエリアの予約**: サイドバーが表示されている間、他のアプリケーション（ブラウザやエクスプローラーなど）がサイドバーと重ならないように、デスクトップの有効領域（Work Area）を自動的に調整します。
- **リサイズ**: サイドバーの境界線をドラッグして幅を調整可能（予定）。

### 3.2 ブラウザ機能
- **WebView2 コンポーネント**: 高速かつ最新のレンダリングエンジンを利用。
- **ナビゲーション**: 戻る、進む、更新、ホーム機能。
- **アドレス入力**: URL または検索ワードによる遷移。
- **お気に入り/クイックリンク**: よく使うサイトへの素早いアクセス（サイドバー上のアイコン等）。

### 3.3 UI/UX デザイン
- **モダンな外観**: Fluent Design やダークモードへの対応。
- **スリムなナビゲーション**: サイドバーとしての特性を活かし、最小限のスペースで操作可能な UI。
- **インタラクティブなフィードバック**: ホバーエフェクトやスムーズなアニメーション。

## 4. 技術的詳細

### 4.1 AppBar 実装
Windows の `SHAppBarMessage` API を呼び出し、以下の状態を管理します。
- `ABM_NEW`: AppBar の登録
- `ABM_POS`: 位置とサイズの予約
- `ABM_REMOVE`: 終了時の登録解除
- `ABM_WINDOWPOSCHANGED`: ウィンドウ位置変更の通知

### 4.2 WebView2 の管理
- `CoreWebView2` の初期化とエラーハンドリング。
- ユーザーデータフォルダ（キャッシュやクッキー）の管理。
- ブラウザの状態（CanGoBack, CanGoForward）のデータバインディング。

## 6. 今後の拡張予定
- **複数タブ/プロファイル**: 異なる用途での使い分け。
- **自動非表示 (Auto-hide)**: マウスホバー時のみ表示するモード。
- **拡張機能連携**: 特定の Web ツールとの統合。
